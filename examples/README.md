# Examples

Runnable demos for each domain. Phase 0 placeholder — real examples land
in Phase 5 alongside the public launch.

## Planned examples

```
examples/
├── physics_quickstart/      # one-cycle physics lab on a small seed
├── editorial_quickstart/    # editorial lab on a sample archive
├── financial_quickstart/    # m_spectro + two-channel on a public dataset
└── custom_domain/           # walkthrough: build a new domain in 30 min
```

Each example is self-contained: docker compose file, sample env, sample
seed, expected output. Designed so a developer evaluating the project in
under an hour can run something real.
